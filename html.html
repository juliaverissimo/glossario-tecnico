tags em HTML

<section>: Agrupar uma sequência de textos.

<section>
<h2> html </h2>
<p> css </p>
</section>

A <section> é usada para definir seções em um documento, agrupando conteúdo relacionado e estruturando a página de forma lógica. Ela é um elemento semântico que ajuda a organizar o conteúdo e pode incluir títulos (<h1> a <h6>) para indicar o tema da seção.

<ol>: Lista ordenada.

<ol>
<li> Primeiro item </li>
<li> Segundo item </li>
<li> Terceiro item </li>
</ol>

O elemento <ol> representa uma lista cujos itens estão organizados em uma ordem específica. Normalmente, cada item é precedido por uma contagem, que pode utilizar números, letras, algarismos romanos ou até símbolos.

<ul>: lista desordenada.

<ul>
<li> Primeiro item </li>
<li> Segundo item </li>
<li> Terceiro item </li>
</ul>

O elemento <ul> define uma lista cujos elementos não seguem uma ordem específica. Nela, a posição de cada item não influencia o significado, e os navegadores costumam exibir cada elemento acompanhado por um marcador, que pode ser um ponto, um círculo ou um quadrado.

<a>: Adicionar um link.

<a href="https://example.com"> Website </a>

O elemento <a> com o atributo <href> cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.

<div>: Contêiner genérico.

<div>
<p> Glossário Técnico </p>
</div>

A tag <div> em HTML serve para criar seções ou agrupamentos de elementos, funcionando como um contêiner genérico. Ela é essencial para organizar o layout da página e aplicar estilos CSS a blocos específicos de conteúdo.
