Tags HTML

<section>: Agrupar uma sequência de textos.

<section>
<h2> html </h2>
<p> css </p>
</section>

A <section> é usada para definir seções em um documento, agrupando conteúdo relacionado e estruturando a página de forma lógica. Ela é um elemento semântico que ajuda a organizar o conteúdo e pode incluir títulos (<h1> a <h6>) para indicar o tema da seção.

<ol>: Lista ordenada.

<ol>
<li> Primeiro item </li>
<li> Segundo item </li>
<li> Terceiro item </li>
</ol>

O elemento <ol> representa uma lista cujos itens estão organizados em uma ordem específica. Normalmente, cada item é precedido por uma contagem, que pode utilizar números, letras, algarismos romanos ou até símbolos.

<ul>: lista desordenada.

<ul>
<li> Primeiro item </li>
<li> Segundo item </li>
<li> Terceiro item </li>
</ul>

O elemento <ul> define uma lista cujos elementos não seguem uma ordem específica. Nela, a posição de cada item não influencia o significado, e os navegadores costumam exibir cada elemento acompanhado por um marcador, que pode ser um ponto, um círculo ou um quadrado.

<a>: Adicionar um link.

<a href="https://example.com"> Website </a>

O elemento <a> com o atributo <href> cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.

<div>: Contêiner genérico.

<div>
<p> Glossário Técnico </p>
</div>

A tag <div> em HTML serve para criar seções ou agrupamentos de elementos, funcionando como um contêiner genérico. Ela é essencial para organizar o layout da página e aplicar estilos CSS a blocos específicos de conteúdo.

<br>: Quebra uma linha.

<p>
Para forçar <br>
uma quebra de linha <br>
Em um texto.</p>

A tag <br> insere uma quebra de linha em um texto, funcionando como um retorno de carro. Ele é especialmente útil em situações em que a divisão de linhas tem importância, como na escrita de poemas ou na formatação de endereços.

<span>: Agrupamento inline.

<p> <span> Glossário Técnico </p>
</span>

A tag <span> serve para reunir e aplicar estilos a conteúdos inline, como trechos de texto ou outros elementos inseridos dentro de um bloco. Por ser um elemento de nível inline, ele não provoca a quebra de linha no fluxo do texto.

<header>: Cabeçalho.

<header>
<h1> Peço silêncio! </h1>
<p> estou estudando. </p>
</header>

A tag <header> representa um contêiner destinado a conteúdo introdutório ou de navegação. Normalmente é posicionado na parte superior da página e pode incluir o logotipo, o título do site, menus e outros elementos importantes. Por ser um recurso semântico, contribui para uma melhor organização do conteúdo, facilitando sua interpretação por navegadores e mecanismos de busca.

<title>: O título do documento.

<title> Glossário técnico </title>

A tag <title> especifica o título de um documento, exibido na barra do navegador ou na aba da página. Ele aceita apenas texto simples, ignorando quaisquer marcações que possam ser inseridas em seu conteúdo.

<footer>: Rodapé.

<footer> Site desenvolvido por aluno. com fins educativos </footer>

A tag <footer> costuma reunir informações como dados de contato, aviso de direitos autorais, links de navegação ou referências relacionadas. Trata-se de um elemento semântico que contribui para a organização do conteúdo e para a oferta de informações complementares sobre a página ou seção.
