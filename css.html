tags em CSS

Padding: Define a área de preencimento.

padding: 10px 50px 30px 0;

No CSS, o padding permite controlar o espaço interno de um elemento. É possível definir o preenchimento individualmente para cada lado (topo, direita, base e esquerda) ou utilizar a forma abreviada para configurar todos os lados em uma única declaração.

Margin: Espaçamento externo.

margin: 70px;

No CSS, as margens podem ser controladas de forma precisa. É possível definir a margem separadamente para cada lado de um elemento (topo, direita, baixo e esquerda) ou usar a propriedade abreviada para configurar todas de uma só vez.

Display: Define exibição de elemento.

display: block;
display: flow-root;

A propriedade display no CSS determina a forma como um elemento HTML é apresentado na página, influenciando sua renderização e o modo como interage com outros elementos. Ela especifica se o elemento será exibido como bloco, em linha ou de outras maneiras, afetando seu posicionamento e o espaço que ocupa.

Justify-content:

div{
display: flex;
justify-content: center;
}

A propriedade justify-content no CSS é usada para controlar a distribuição do espaço entre os itens de um contêiner, ao longo do eixo principal em contêineres flexíveis e do eixo inline em contêineres de grade e multicolunas.

Position: Posição do elemento.

position: absolute;
inset-inline-start: 40px;
inset-block-start: 40px;

A propriedade position do CSS determina a forma como um elemento é posicionado dentro de um documento. Ela é utilizada junto com as propriedades top, right, bottom e left para definir a localização exata do elemento na página.
