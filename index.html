Glossário Técnico

Um glossário técnico é uma lista na qual escrevemos termos junto de suas definições. Como: Facilitar a compreenção, definir termos e auxiliar na tradução dos termos. De uma determinada área ou campo de conhecimento. Sendo usado como uma ferramenta de clareza e suporte, dentro de um setor técnico.Neste glossário, iremos esclarecer Tags HTML, CSS e JAVA SCRIPT.

Em HTML, tags são comandos usados para marcar e estruturar elementos em uma página da web. Elas indicam ao navegador como exibir e organizar o conteúdo, como texto, imagens e links. As tags são escritas entre colchetes angulares <> e geralmente aparecem em pares: uma tag de abertura e uma tag de fechamento, embora algumas tags sejam auto fecháveis.

Em CSS, termos como seletor, propriedade, valor, regra, bloco de declaração, e folha de estilos são fundamentais para entender como a linguagem funciona. O CSS (Cascading Style Sheets) é usado para definir a aparência de documentos HTML, separando o conteúdo da sua apresentação visual.

JavaScript é uma linguagem de programação de alto nível, amplamente utilizada para adicionar interatividade e funcionalidades dinâmicas a páginas web. É uma linguagem interpretada, o que significa que o código é executado diretamente no navegador, sem a necessidade de compilação.
